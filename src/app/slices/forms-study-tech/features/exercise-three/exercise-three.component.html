Descrição do Exercício 3

Neste exercício, você vai criar um formulário dinâmico de cadastro de tipos de grãos utilizando Reactive Forms no Angular. O objetivo é capturar informações detalhadas sobre os grãos cultivados. O formulário deve conter 4 campos obrigatórios para cada tipo de grão, e os dados preenchidos devem ser exibidos em tempo real na tela.

Campos para o Formulário:

  • Nome do Grão (grainName): Campo de texto para o nome do grão.
  • Quantidade em Toneladas (quantity): Campo numérico para a quantidade de grãos em toneladas.
  • Data da Colheita (harvestDate): Campo de data para a data da colheita.
  • Região de Produção (region): Campo de texto para a região onde o grão foi produzido.

Objetivos:

  • Criar um formulário que permita adicionar múltiplos tipos de grãos dinamicamente.
  • Validar os campos obrigatórios para cada grão adicionado.
  • Exibir os valores do formulário em tempo real no formato JSON.
  • Impedir a submissão do formulário caso algum campo obrigatório não esteja preenchido.
  • Permitir adicionar e remover tipos de grãos conforme necessário.
