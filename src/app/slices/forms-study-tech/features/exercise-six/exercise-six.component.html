Descrição do Exercício 6

Neste exercício, você implementará um formulário que captura a seleção de uma Regional e, com base nela, atualiza as opções disponíveis para seleção de Filiais. Utilizando Template-Driven Forms no Angular, o objetivo é entender como utilizar RxJS para reatividade e escrita declarativa em formulários.

Objetivos do Exercício:

  • Criar o Formulário de Seleção:
    • O formulário deve permitir que o usuário selecione uma Regional.
    • Com base na Regional selecionada, o formulário deve atualizar automaticamente as opções de Filial disponíveis para seleção.
  • Integração com RxJS:
    • Utilizar RxJS para observar a seleção da Regional e, de forma reativa, atualizar a lista de Filiais disponíveis.
    • Implementar uma escrita declarativa para lidar com as mudanças no formulário.
  • Integração com Signals:
    • Utilizar signals para observar a seleção da Regional e, de forma reativa, atualizar a lista de Filiais disponíveis.
    • Implementar uma escrita declarativa para lidar com as mudanças no formulário.
  • Submissão do Formulário:
    • O botão de submissão deve estar desativado até que uma Regional e uma Filial sejam selecionadas.
    • Ao submeter o formulário com sucesso, os dados devem ser processados (por exemplo, exibidos no console ou enviados a um serviço).

Exemplo de Comportamento:

  • Ao selecionar uma Regional, a lista de Filiais é atualizada automaticamente para exibir apenas as Filiais associadas àquela Regional.
  • O formulário só pode ser enviado quando tanto a Regional quanto a Filial estiverem selecionadas.
