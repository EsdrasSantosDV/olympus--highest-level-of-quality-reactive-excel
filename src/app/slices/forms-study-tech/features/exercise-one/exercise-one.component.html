Descrição do Exercício 1

Neste exercício, você vai criar um formulário simples de cadastro de fazendas utilizando Template-Driven Forms no Angular. O objetivo é capturar informações básicas sobre a fazenda. O formulário deve conter 5 campos obrigatórios, e os dados preenchidos devem ser exibidos em tempo real na tela.

Campos para o Formulário:

  • Nome da Fazenda (farmName): Campo de texto para o nome da fazenda.
  • Proprietário (ownerName): Campo de texto para o nome do proprietário da fazenda.
  • Área Total (hectares) (totalArea): Campo numérico para a área total da fazenda em hectares.
  • Localização (location): Campo de texto para a localização da fazenda.
  • Data de Fundação (foundationDate): Campo de data para a data de fundação da fazenda.

Objetivos:

  • Criar um formulário que capture os dados acima.
  • Validar os campos obrigatórios.
  • Exibir os valores do formulário em tempo real no formato JSON.
  • Impedir a submissão do formulário caso algum campo obrigatório não esteja preenchido.
