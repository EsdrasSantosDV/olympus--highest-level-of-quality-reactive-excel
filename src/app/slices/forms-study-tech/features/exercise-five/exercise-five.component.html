Descrição do Exercício 5: Controle Personalizado com Control Value Accessor

Neste exercício, você implementará um formulário de cadastro de produto utilizando Angular, onde será possível interagir com um controle personalizado que representa um campo de texto simples. O objetivo é compreender como integrar controles personalizados em formulários utilizando o Control Value Accessor, tanto em Template-Driven Forms quanto em Reactive Forms.

Objetivos do Exercício:

  • Criar o Formulário de Cadastro de Produto:
    • O formulário deve ter um campo obrigatório, onde o usuário possa inserir uma descrição do produto.
  • Integração com Controle Personalizado:
    • Utilizar um controle personalizado (exemplo: um campo de texto customizado) para capturar a descrição do produto.
    • Implementar esse controle personalizado de forma que ele possa ser utilizado tanto em Template-Driven Forms quanto em Reactive Forms utilizando o Control Value Accessor.
  • Validações Implementadas:
    • Validar que o campo de descrição não esteja vazio.
    • Exibir uma mensagem de erro se o campo estiver vazio.
  • Submissão do Formulário:
    • O botão de submissão deve estar desativado até que o formulário seja considerado válido.
    • Ao submeter o formulário com sucesso, os dados do produto devem ser processados (por exemplo, exibidos no console ou enviados a um serviço).
